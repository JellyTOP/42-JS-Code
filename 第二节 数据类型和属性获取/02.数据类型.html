数据类型 就是数据种类

1.数值型 number 包括小数点和整数!在js当中小数运算是不精确的 我们不能用它当判断

2.字符串 string 字符连成了串 可以用"" ''包裹,此时字符串不能换行;``反引号包裹(模板字符串)

3.布尔值 boolean true/false

4.function 没有名的函数,称之为匿名函数,不能莫名其妙的出现一个匿名函数 有名字的函数,成为具名函数 具名函数的使用 调用使用函数名+()

5.对象 object 简单写法为{},实际上空的一个对象使用过new得到的,new Object

6.数组 object 数组中可以存放任何数据类型 简写方法[数据,数据,数据],空数组为[],是由new Array创造的

7. null object 本质是一个空指针,解释为在找对象的时候没找到(拿到了一个安慰奖)null

undefined undefined 申明了但是并未赋值
